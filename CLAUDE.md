## Game Design Skills

When doing design, evaluation, or planning work on this project, read and follow the appropriate skill from ~/gamedev-skills/skills/:

- Design docs & mechanic specs → gamedev-design-doc
- Story, characters, lore → gamedev-narrative
- Level design & pacing → gamedev-level-design
- Economy, balance, probability → gamedev-systems-balance
- Visual style & art pipeline → gamedev-art-direction
- Sound & music → gamedev-audio
- Code structure & patterns → gamedev-code-architecture
- HUD, menus, game feel → gamedev-ui-ux
- Testing & QA → gamedev-qa-testing
- Scope, milestones, planning → gamedev-production

Always read the full SKILL.md plus any templates and references it points to before starting work. Shared design principles are at ~/gamedev-skills/shared/.
